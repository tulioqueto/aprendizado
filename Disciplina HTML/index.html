Mas trabalho é trabalho. Então esta página vai contar um pouco da minha história com o HTML

Então vamos começar a aprender para que serve cada tag. E pensar que meu primeiro contato com o HTML, nem lembro mais que versão era, mas lembro muito bem de ter feito mei primeiro site com um programa chamado Aracnofilia. Como se utilizava apenas de código, passei a ver código fonte dos sites no navegador para entender a estrutura de um site. Até aprendi a montar e carregar um .gif animado e montar uma tabela. O ano era 1999, era "The Matrix!" no cinema e a ascensão do Flash nas redes. E precisavamos instalar o Java na máquina. Gente, me senti velho agora.

Muita coisa mudou até ver o HTML5 (e o tal de CSS, tomei uma surra quando vi pela primeira vez) mas de verdade, só não entendo porque em HTML o negrito é STRONG e não o Bold como no Word!

Fazer o Site foi de boa, era azul no fundo, tinha uma tabela legalzinha, e tinha hospedagem grátis em um site chamado hpg (Home Page Grátis) que o link era www.hpg.com.br, que ante que me pergunte onde está o link, foi descontinuado, e nem me preocupei em salvar, não achei que iria encarar no futuro, a área de tecnologia. Foi neste site que aprendi a usar o FTP para baixar e subir arquivos.

Fiz algumas brincadeiras a mais com HTML, mas fui para área de biologia e imunologia. Quem diria que meus conhecimentos de html me permitissem criar um site de um evento de imunologia em 2006. Sim, fiz o site, que foi hospedado no antigo mhx, mas como eu que teria que pagar para manter o site, ele sumiu em 2007. Mas resolvi fuçar no Wayback Machine do Internet Archive e não é que eu achei este meu site?

O site do I CONIP foi meu xodózinho, pena que não tenho mais as imagens que montei (ficou tudo no laboratório que era fonte dos arquivos).

Embora tenha deixado de fazer sites, nunca deixei de brincar com html as vezes. Até hoje posso dizer que já usei diversos programas para montar sites:

  1. Aracnofilia
  2. Frontpage
  3. Dreamweaver
  4. Bloco de texto
  5. Visual Studios Code

Mas me aventurei também em fazer algumas coisas em sites on line como:

  • Wix
  • Sites do Google

Recentemente descobri o HTML 5, e a desvantagem de tentar montar um menu com figuras, gente, que desespero, acho que vou ficar só no texto e tamanho padrão para o celular e desktop.

Bom, esta é minha história com o HTML. Espero que tenham gostado!
